This code contains all necessary packages and functions to replicate statistics and graphs pertaining to the manuscript entitled: "Skeletal muscle properties in long COVID and ME/CFS differ from those induced by bed rest" (Charlton et al)

Manually input working directory at line 
Manually input date at line 
Ensure that the file "Manuscript_data_clean_230525.xlsx" is located in the working directory

To run code: download excel file to working directory, input working directory and date, and run all.

Commented areas are not necessary for replication of statistics or graphs. These can be uncommented for either assessing normality of data or comparing correlations.
